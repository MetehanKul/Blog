About Me

I work on projects involving machine learning, natural language processing (NLP), and data visualization. My goal is to become a skilled software developer and realize my dreams in the tech industry. Developed a production analysis tool using Python for data cleaning and EDA. Created interactive Power BI dashboards to track KPIs like production efficiency, scrap rates, and profitability. The project offered key insights for optimizing processes and improving performance.

The application will allow users to input various attributes of the vehicle (such as model, year, mileage, engine size, etc.) and will use machine learning algorithms to estimate the price based on the provided data.
